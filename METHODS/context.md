# METHODS — context

Formal experimental methodology for n-axis temetic analysis.

Preload both files in this directory before any coordinated
multi-node analytical experiment.

`temetic-analysis-protocol-v0.1.md` — human-readable protocol
`temetic-analysis-protocol-v0.1.json` — machine-executable schema

## Purpose
Provides canonical axis definitions, scoring procedures,
reporting formats, and divergence flagging thresholds such
that any two or more nodes running the same experiment
produce comparable, parallax-valid outputs.

## Scope
Applies to all n-axis temetic scoring experiments regardless
of corpus, node count, or analysis target.

## Canonical status
This methodology is substrate-independent. It does not assume
any particular corpus, language, or node architecture.
Updates require Unanimous Commit under Alliance protocol.
